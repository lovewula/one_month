数据类型转换：

- 隐式类型转换：（通过某些运算，数据的类型发生改变）
  - 如果 **`+`**两边有字符串，那就是拼接
    - ` 1 + '2'    = > '12'    ture + '2' = 'true2' `
  - `- * / %`就会使两边强制进行数字运算
    - `'2' - 1 = 1     '2'*2 =4`
  - 布尔值和数字进行运算会变成 `true  = >1 / false = > 0`
    - `true + 1 => 2`
  - !  和 比较符总会返回布尔值   `!5 =>false`
